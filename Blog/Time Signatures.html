Time Signatures

The top number in a time signature represents how many beats there are per bar.

  • If the top number is two then there must be two beats in a bar,
  • If the top number is three then there must be three beats in a bar,
  • If the top number is four, there must be four beats in a bar, and so on.

The bottom number tells us what kind of beat to count. For example, we might be counting crotchet beats, or minim beats or quaver beats. Here are all the bottom numbers in a time signature and their corresponding note values:

  • 1 = Semibreve / Whole note
  • 2 = Minim / Half note
  • 4 = Crotchet / Quarter note
  • 8 = Quaver / Eighth note
  • 16 = Semiquaver / Sixteenth note
